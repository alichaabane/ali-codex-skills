# Contributing

Ali Codex Skills is a modular engineering skill repository for ChatGPT Codex. Keep contributions small, focused, and aligned with the existing skill structure.

## Guidelines

- Give each skill one clear responsibility.
- Keep `SKILL.md` files concise and procedural.
- Avoid duplicating guidance across skills.
- Prefer references over growing a skill into a large document.
- Keep commits focused on one logical change.
- Keep each pull request scoped to one coherent improvement.

## Contribution Scope

- Add or refine skills only when the responsibility boundary is clear.
- Prefer improving the nearest existing skill before creating a new one.
- Keep public project docs aligned with the current repository structure.
